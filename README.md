k8s-dev
=======

このリポジトリは、Kubernetes環境におけるAI/LLMワークロード（大規模言語モデルなど）の開発、デプロイ、およびベンチマーク検証を行うための構成やスクリプトを管理しています。

主な要素:
- **vLLM / KubeAI / Ray**: LLMのサービング環境
- **ComfyUI**: 画像生成ワークロード
- **Bench**: AIPerf等を用いたモデル推論のパフォーマンスベンチマーク
- **Argo Workflows**: ワークフロー管理
- **Monitor**: オブザーバビリティ・監視設定
- **GPU**: GPUリソースの管理・設定
